MQT_RenkoPatterns

Automated Renko/Range Pattern Recognition Strategy

MQT_RenkoPatterns is a fully automated NinjaTrader 8 strategy designed for Renko and Range bar types. It detects pullback-rally reversal patterns, executes entries with precision stop-loss and target management, includes optional trailing stops, and features integrated ChartTrader buttons for manual control. Built with robust order management, session filtering, and optional trade markers with customizable labels.

Key Features

Pullback-Rally Pattern Logic

Detects reversal patterns by counting consecutive bars in opposite directions. Configurable min/max pullback bars and min rally bars for precise pattern recognition.

Brick-Based Stops & Targets

Stop-loss and profit targets calculated as multiples of the Renko/Range brick size for consistent risk/reward across all market conditions.

ChartTrader Integration

Three custom buttons (Buy Long, Sell Short, Close One) added to ChartTrader panel for manual entries and exits without leaving the chart.

Top Toolbar Strategy Toggle

Strategy-ON/OFF button in the chart window's top toolbar (MainMenu) for instant enable/disable. Green when enabled, red when disabled.

Session Time Filtering

Define start/end times for trading window. Strategy only takes signals during active window and can auto-flatten positions at end time.

Optional Trailing Stops

Activate trailing stops after a configurable trigger distance (in ticks) is reached, then trail at a specified distance to lock in profits.

Reversal Mode

Enable reversals to automatically exit current position and enter opposite direction on counter-signal. Disable for directional-only trading.

Signal Inversion

Invert all signals (long becomes short, short becomes long) for contrarian strategies or testing alternative logic.

Max Primary Positions

Limit the number of concurrent primary positions to control risk and avoid over-trading during choppy conditions.

ChartTrader Quantity Sync

Optionally sync trade quantity from ChartTrader's quantity selector for dynamic position sizing without reloading the strategy.

Customizable Trade Markers

Draw entry arrows (up/down) and exit dots on chart with configurable colors, sizes, and optional text labels showing qty, price, or both.

Robust Order Management

Handles out-of-sequence events, order cancellations, and position updates with internal order tracking and timer-based execution reconciliation.

How It Works

1. Pattern Detection

On each bar close (or price change), strategy counts consecutive bars in opposite directions to identify pullback-rally patterns matching configured parameters.

2. Signal Generation

When pullback bars (min/max) are followed by rally bars (min), a primary signal is generated. Signal can be inverted if InvertSignals is enabled.

3. Entry Validation

Checks if another entry is allowed (max primary positions, session window, strategy enabled). If reversal mode is on and opposite position exists, initiates reversal.

4. Market Entry

Submits market order for configured quantity (from parameter or ChartTrader). Entry order name includes "Manual" if triggered by button click.

5. Stop & Target Placement

On fill, calculates stop-loss (entry ± brick × StoplossBars) and profit target (entry ± brick × TargetBars). Submits stop-market and limit orders.

6. Trailing Stop (Optional)

If TrailStopTriggerTicks > 0 and price moves favorably, activates trailing stop at TrailStopDistanceTicks from current price.

7. Exit Management

Position exits on stop-loss hit, target hit, trailing stop hit, manual "Close One" button, or end-of-session flatten (if enabled).

8. Trade Markers (Optional)

Draws entry arrows and exit dots on chart with optional text labels showing quantity, price, or both. Fully customizable colors and sizes.

FAQ

Does this work on time-based bars?

The strategy is designed for Renko and Range bar types. It will run on time-based bars but the brick-based stop/target logic may not be optimal. A warning is printed if bar type is not Renko/Range.

Can I use this for live trading?

Yes — designed for real-time execution with robust order management, session filtering, and manual override buttons. Backtest thoroughly and monitor the strategy toggle before going live.

How do I enable trailing stops?

Set TrailStopTriggerTicks > 0 (e.g., 10 ticks). Once position is in profit by that amount, trailing stop activates at TrailStopDistanceTicks from current price and trails as price moves favorably.

What happens at end of session?

If FlattenAtEndTime is enabled, strategy exits all positions at EndTime. If disabled, positions remain open into next session (use with caution for overnight risk).

Can I reverse from long to short automatically?

Yes — enable Reversals parameter. On counter-signal, strategy cancels all orders, exits current position, then enters opposite direction. Disable for directional-only trading.

How do I hide trade markers?

Set PlotTradesOnChart to False. All entry/exit arrows and dots will be hidden. Re-enable to show markers again.

What is the "Close One" button for?

Manually exits the first active position (useful if you have multiple positions from MaxPrimaryPositions > 1). Cancels its stop/target and submits market exit for that position's quantity.

Can I customize marker colors and sizes?

Yes — all marker colors (long/short arrows, exit dots, text labels) and sizes (arrow/dot) are fully customizable in the Visual settings group.

How does the top toolbar toggle work?

A "Strategy-ON" (green) or "Strategy-OFF" (red) button is added to the chart window's top toolbar. Click to toggle. When OFF, automated signals are blocked and manual buttons print a "blocked" message.

What if I want to test contrarian logic?

Enable InvertSignals to reverse all signals (long becomes short, short becomes long). Useful for testing alternative strategies or contrarian approaches.
